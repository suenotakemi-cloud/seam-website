# SalonPro — 美容師・美容室向け BtoB美容卸EC（フロント土台）

仕様書（`../01〜04`）に基づく、プロ専用・会員制 美容商材ECのフロントエンド土台。
試作スクショの「商品一覧」の見た目を、静的HTML/CSS/JSで実データ・実動作付きに再現したもの。
全画面で使い回せるデザインシステム（トークン → 共通コンポーネント → ページ）として構成。

## 技術構成
- 静的 HTML / CSS / バニラ JS（ビルド不要・即プレビュー・どこでも配信可）
- フレームワーク・依存なし。`<script>` 直読み込み。
- データは配列管理（`assets/js/data.js`）。実APIへ差し替え可能な形。

## ディレクトリ
```
salonpro/
├─ index.html                商品一覧ページ（シャンプー）
├─ assets/
│  ├─ css/
│  │  ├─ tokens.css          デザイントークン（色/余白/角丸/影/タイポ）← 全体のトーンはここ
│  │  ├─ base.css            リセット & 土台
│  │  ├─ components.css      共通UI（ヘッダー/告知/ピル/カード/カートバー/タブ/ドロワー）
│  │  └─ listing.css         一覧ページ固有
│  └─ js/
│     ├─ data.js             商品・カテゴリのマスタ（サンプル）
│     ├─ store.js            カート/お気に入りの状態管理（localStorage + 購読）
│     ├─ components.js        描画ヘルパー（アイコン/価格/在庫/プレースホルダ/カード）
│     └─ app.js              一覧ページ制御（描画/絞り込み/並び替え/検索/同期）
└─ README.md
```

## 実装済みの機能（商品一覧）
- 商品カード描画（ブランド/商品名/価格/在庫/数量ステッパー/カート）
- バッジ：人気・新商品・残りわずか
- 在庫状態の固定語彙：在庫あり / 残りわずか / 取寄せ / 入荷待ち（色＋アイコン＋テキスト併用）
- カート投入 → 下部カートバー（点数・合計税抜）＋ヘッダー/タブのバッジ同期（localStorage永続）
- お気に入りトグル（永続）
- カテゴリ切替（クイック導線＋ドロップダウン）
- 並び替え（人気/新着/価格昇順・降順/名前）
- ブランド絞り込み（ドロップダウン）
- 絞り込みドロワー（在庫・納期／当日出荷／価格帯、件数プレビュー、適用中チップ）
- 全カテゴリ横断のキーワード検索（ドロワー＋人気キーワード）
- 表示切替（グリッド／リスト）
- レスポンシブ：スマホ2列（フッター縦積み）→ タブレット3列以上（横並び・コンテナクエリ）
- 実画像が無くても崩れない上質なSVGプレースホルダ（ボトル/詰替/箱/チューブ/ジャー）

## プレビュー方法
このリポジトリは静的配信できる。ローカルで見るには salonpro/ を配信するだけ：
```
cd salonpro && python3 -m http.server 8000
# → http://localhost:8000
```
※ Claude Codeのプレビューはサンドボックスの都合で `/tmp` 配下のみ配信可。
　 そのため `/tmp/salonpro-preview` へ実体コピーして配信している（port 3100）。
　 ソース変更後の同期コマンド：
```
cp -R "/Users/suenotakemi/Downloads/美容室向けBtoB美容卸EC_一式/salonpro/." /tmp/salonpro-preview/
```
