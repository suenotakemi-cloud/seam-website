# 🎨 撮影/生成 推奨指針 (全ページ共通)

### トーン・マナー
- **色温度**: 暖色寄り (warm white 2700–3200K)
- **彩度**: ややくすみ気味 (Aman Resort / Aesop 級の上品さ)
- **背景**: ライムウォッシュ壁 / 木目什器 / 籐 / リネンを基調
- **被写体**: 「顔よりも手と手元」「全身よりも所作」を優先 (お客様像を狭めない)

### モデル/人物
- **ヘッドスパ系**: 地中海系モデル (既存spa-v2と整合) も可、日本人モデルも併用
- **ヘアサロン系**: 多国籍を意識 (インバウンド対応)
- 顔のクローズアップは最小限、手元・髪・所作中心

### 動画
- **長さ**: 8〜12秒、ループ前提
- **音声**: なし (autoplay muted)
- **動き**: 超スロー、ASMR的、突然の動きなし

### 静物・商品
- **背景**: ベージュ/オフホワイト/木目
- **影**: ソフトでロングシャドウ、ハードな影は厳禁
- **数**: 1点 or 3点 (2点・4点は構図が締まらない)
