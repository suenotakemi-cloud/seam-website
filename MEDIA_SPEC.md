### A. ライティング (全媒体共通)

| 項目 | 仕様 |
|------|------|
| 色温度 | **2900–3200 K** (warm white, 真っ白厳禁) |

### B. 構図ルール

| 項目 | 仕様 |
|------|------|
| 余白 | 被写体周囲の余白を **画面の40%以上** 確保 |
| 構図 | 三分割法、または **左寄せ・右寄せ非対称構図** (中央構図禁止) |
| 視線 | 顔のクローズアップは最小限、**「手・所作・後ろ姿・髪のライン」**を主役に |
| 被写体 | 商品は **1点 / 3点 / 5点** (2点/4点禁止) |

### C. カラー・現像 (LR/Capture One)

| 項目 | 数値の目安 |
|------|----------|
| 露出 | 0 〜 +0.3 EV (明るめ) |
| ハイライト | -15 〜 -25 |
| シャドウ | +20 〜 +30 (持ち上げ) |
| 彩度 | -8 〜 -12 (くすませる) |

| トーンカーブ | ベージュ/グレージュにシフト、純白(255)を避け **252,248,242 上限** |

### D. 被写体モデル

| 項目 | 仕様 |
|------|------|
| 国籍 | **多国籍** (日本/欧州/中華圏/韓国を意識的に分散) |
